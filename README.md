# Minha To-Do List

## Sobre o projeto
Este é um programa simples de **lista de tarefas** (To-Do List) feito em Python.  
Ele permite adicionar, remover e marcar tarefas como concluídas.

## Como executar

### Pré-requisitos
- Python 3 instalado no computador

### Passos
1. Baixe os arquivos
2. Abra o terminal na pasta do projeto
3. Execute: `python main.py`

## Funcionalidades
-  Adicionar nova tarefa
-  Marcar tarefa como concluída
-  Remover tarefa
-  Listar todas as tarefas
